Last.fm widget for websites

Published: 25 Aug 2011
Note: This is an archived post. Information may not be relevant now.

This widget shows last played tracks from last.fm for a user. It also shows ‘Now Playing’ track if available.

Download: https://github.com/xtranophilist/lastfm-widget/zipball/master

Integration:

Insert: <table id="last_fm_tracks"></table> where you want the script to write the track lists.

Add:

<script type="text/javascript" src="last_widget.js"></script>  
<script type="text/javascript">  
loadLastFMTracks('xtranophilist');  
</script>

just before the end of the body tag </body> Don’t forget to replace xtranophilist with your username. If you are using ID for the <table> other than last_fm_tracks, make sure you pass the ID string as the second parameter for the loadLastFMTracks() method.
