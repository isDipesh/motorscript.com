IMDB to GetGlue Bookmarklets

Published: 28 Jan 2013
Updated: 19 Dev 2013
Note: This is an archived post. Information may not be relevant now.

You can use these bookmarklets to open GetGlue page of a movie or TV show from its IMDB page. I created these because I thought it would be a huge timesaver and searching in GetGlue isn’t pleasant at all.

For GetGlue old version users,
I2OGG

For ‘new GetGlue’ users,
I2GG

Installation Instructions:
Drag your preferred link to your bookmark toolbar.

The script tries to guess the URL for GetGlue from movie/tv-show title and director’s name from IMDb’s page. It may not work for all pages as GetGlue’s URL creation may not be consistent (The only inconsistent one I found was for The Godfather).

If you have any suggestions, feel free to comment.

The source code is available from this gist.
